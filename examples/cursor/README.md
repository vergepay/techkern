# Cursor

Drop `mcp.json` into `~/.cursor/mcp.json` (global) or `<project>/.cursor/mcp.json` (per-project).

Reload the MCP panel — `techkern` will appear with all 10 tools available.

Tips:
- Pin techkern in the agent settings so it's always live in chat
- Hosted URL mode (`url: "https://api.techkern.xyz/mcp"`) skips the `npx` install on cold start
