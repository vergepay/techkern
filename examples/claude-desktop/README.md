# Claude Desktop

Drop the snippet from `claude_desktop_config.json` into your Claude Desktop config:

- macOS: `~/Library/Application Support/Claude/claude_desktop_config.json`
- Windows: `%APPDATA%\Claude\claude_desktop_config.json`

Restart Claude. Your conversation now has tools like `query_solana_swaps`, `query_pump_fun_launches`, and the rest.

Try:

> "Show me the 5 largest USDC -> SOL swaps in the last hour."
>
> "Has the dev for mint 7xK... ever rugged a token before?"
>
> "List KOL wallets with >$50k 30-day PnL that bought $PEPE in the last 24h."
